Fields

bins: Vec<BTreeSet<Frame>>

The bins of free blocks, where bin i contains blocks of size 2^i. Uses BTreeSet to store the blocks in each bin, for efficient buddy location. The Frame stored for each block is the first frame in that block - the end frame can be calculated from the start frame and the order of the bin the block is in.
