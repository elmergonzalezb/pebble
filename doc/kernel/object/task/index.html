TaskStack

Represents the layout of a task's usermode or kernelmode stack. A slot is allocated (contiguous from slot_bottom to top), but only a portion of it may initially be mapped into backing memory (contiguous from stack_bottom to top). A stack can be grown by allocating more backing memory and moving stack_bottom down towards slot_bottom.
