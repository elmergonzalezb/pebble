Fields

bins: Vec<BTreeSet<PhysicalAddress>>

The bins of free blocks, where bin i contains blocks of size 2^i. Uses BTreeSet to store the blocks in each bin, for efficient buddy location. Each block is stored as the physical address of the start of the block. The actual frames can be constructed for each block using the start address and the order of the block.
