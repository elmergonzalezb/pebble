Create a PageTableImpl from a Frame that already contains a P4. This is very unsafe because it assumes that the frame contains a valid page table, and that no other PageTableImpls currently exist that use this same backing frame (as calling mapper on both could lead to two mutable references aliasing the same data to exist, which is UB).
