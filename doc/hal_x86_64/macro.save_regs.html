macro_rules! save_regs {
    () => { ... };
}

Macro to save the scratch registers. In System-V, rbx, rbp, r12, r13, r14, and r15 must be restored by the callee, so Rust automatically generates code to restore them, but for the rest we have to manually preserve them. Use restore_regs to restore the scratch registers before returning from the handler.
